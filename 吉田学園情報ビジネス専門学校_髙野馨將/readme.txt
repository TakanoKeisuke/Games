◆操作方法◆
※ジョイパッド推奨です

「キーボード」
・方向キー(矢印キー)		→	プレイヤーの移動
・Z,C,M,Nキー 	 			→	カメラの旋回
・ENTERキー					→	ギミック(ジャンプ台)を使用
・SPACEキー 				→	ジャンプ

「ジョイパッド」
・左スティック				→	プレイヤーの移動
・右スティック				→	カメラの旋回
・Bボタン					→	ジャンプ
・X,Yボタン					→	ギミック(ジャンプ台)を使用
・Aボタン 					→	長押しでダッシュ
・右スティック押し込み 		→	プレイヤーの正面にカメラを向ける
